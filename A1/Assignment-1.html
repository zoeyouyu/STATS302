Q4. Should we compute the principal components of this dataset based on the correlation matrix or covariance matrix (ie with or without scaling the data)? Explain your reasoning.
Answer:

We dont need to scale the data since it is already on the same scale. Days with less variability are just less important. So we compute the principle components of this dataset based on the covariance matrix.
